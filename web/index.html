ℹ Bulk Import from Data Export

  2. Request your data in JSON format
  3. Wait for Instagram to prepare your download (may take up to 48 hours)
  4. Download and extract the ZIP file
  5. Find following.json in connections/followers_and_following/
  6. Click "Import File" below and select that JSON file
